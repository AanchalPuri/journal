Training a Convolutional Neural Network Using Keras¶

A very common problem in computer vision is recognizing hand-written digits. The images of numerals are commonly used by data scientists and machine learning experts to train supervised learning models that specialize in decoding human handwriting. This is a classic problem that is often used in exercises and documentation. In this demo, we will show you how to train a convolutional neural network to classify hand-written digits using Keras in TensorFlow.

Note: some of the code cells in this notebook may take a while to run

Define the ML Problem¶

We will implement a convolutional neural network to solve a handwriting recognition problem. The neural network will classify a hand-written digit.

Define the Label¶

We will work with the MNIST data set, a famous collection of images used for handwriting recognition. It contains labeled images of handwritten digits from 0 to 9. Therefore, the label is a digit from 0 and 9. This is a multiclass classification problem.

Identify Features¶

Each example corresponds to one hand-written image. The features will be comprised of numerical feature vectors (an n-dimensional array) that contain grey-scale pixel values that range from 0 to 255.

    plt.show()

Prepare the Data¶

Let's now prepare our data to be suitable for a CNN.

Scale the Data¶

Our MNIST data is raw data containing pixel values between 0 and 255. Neural networks process inputs using small weight values, and inputs with large integer values can disrupt or slow down the training process. Therefore, it is a good practice to normalize the pixel values so that each pixel has a value between 0 and 1. This can be done by dividing all pixels values by the largest pixel value; that is 255.

The code cell below normalizes the pixel values in X_train and X_test to be between 0 and 1 by dividing all feature values by 255.0.

In [10]:
X_train, X_test = X_train / 255.0, X_test / 255.0

Reshape the Data¶

A CNN in Keras requires a 4-dimensional array as input in the form: (num_examples, image_dimension_X, image_dimension_Y, num_channels).

Since grayscale has only one color channel, every example in X_train would have the shape (28, 28, 1). X_test should have the same dimensions.

The code cell below uses the NumPy reshape() function to:

  1. reshape every example in X_train to have the shape (num_examples_X_train, 28, 28, 1).
  2. reshape every example in X_test to have the shape (num_examples_X_test, 28, 28, 1).
In [11]:
X_train = np.reshape(X_train, (X_train.shape[0], 28, 28, 1))
X_test = np.reshape(X_test, (X_test.shape[0], 28, 28, 1))

Implement a Convolutional Neural Network¶

Recall that the steps to implement a network model using Keras are:

  1. Construct the Neural Network:
    1. Define the model structure.
    2. Define the optimization algorithm, i.e., the variant of gradient descent you want to use.
    3. Define the loss function to minimize.
    4. Compile the model.

  2. Train the neural network:
    "Fit" the model to the training data for $M$ epochs.

  3. Evaluate the model's performance.

We will follow these same steps to implement a convolutional neural network. We will walk you through these steps.

Part 1. Construct the Convolutional Neural Network¶

Step 1. Define Model Structure¶

We will create our convolutional neural network structure. A CNN typically contains three main types of layers: convolutional layers, pooling layers, and fully connected (dense) layers. When constructing a convolutional hidden layer, we will compose a 2D convolution, followed by a batch normalization, followed by an activation function.

  • As before, we will use the Keras Sequential class to group a stack of layers. This will be our CNN model object. For more information, consult the Keras online Sequential class documentation.

The code cell below creates the CNN structure (Note that there are different ways one can choose to construct a CNN in Keras). We will create an input layer, five hidden layers and an output layer:

  • Input layer: The input layer will have the input shape corresponding to the number of features.
  • Hidden layers: We will create five hidden layers:
    • Four hidden layers will be convolutional layers. They will be comprised of a 2D convolution, followed by a batch normalization, followed by an activation function. In this case, the activation function of choice is ReLU.
    • One hidden layer will be a pooling layer. We will add a layer that uses Global Average Pooling. This is a pooling operation designed to replace the final fully connected layer in classical CNN.
  • Output layer: The output layer will have a width of 10.
In [12]:
# Create CNN model object

cnn_model = keras.Sequential()

# Construct the input layer and add it to the model object: 
input_layer = keras.layers.InputLayer(input_shape=(28, 28, 1))
cnn_model.add(input_layer)

# Construct the first convolutional layer and add it to the model object:
conv_1= keras.layers.Conv2D(16, 3) 
batchNorm_1 =  keras.layers.BatchNormalization()
relu_1 = keras.layers.ReLU()                             
cnn_model.add(conv_1)
cnn_model.add(batchNorm_1)
cnn_model.add(relu_1)

# Construct the second convolutional layer and add it to the model object:
conv_2= keras.layers.Conv2D(32, 3) 
batchNorm_2 =  keras.layers.BatchNormalization()
relu_2 = keras.layers.ReLU()
cnn_model.add(conv_2)
cnn_model.add(batchNorm_2)
cnn_model.add(relu_2)

#  Construct the third convolutional layer and add it to the model object:
conv_3= keras.layers.Conv2D(64, 3) 
batchNorm_3 = keras.layers.BatchNormalization()
relu_3 = keras.layers.ReLU()
cnn_model.add(conv_3)
cnn_model.add(batchNorm_3)
cnn_model.add(relu_3)

# Construct the fourth convolutional layer and add it to the model object:
conv_4= keras.layers.Conv2D(128, 3)
batchNorm_4 =  keras.layers.BatchNormalization()
relu_4 = keras.layers.ReLU()
cnn_model.add(conv_4)
cnn_model.add(batchNorm_4)
cnn_model.add(relu_4)

# Construct the pooling layer and add it to the model object:
pooling_layer = keras.layers.GlobalAveragePooling2D()
cnn_model.add(pooling_layer)

# Construct the output layer and add it to the model object:
output_layer = keras.layers.Dense(units=10) 
cnn_model.add(output_layer)

# Print model summary
cnn_model.summary()

Let's break this down. Our network is comprised of an input layer, five hidden layers (four convolutional layers and a pooling layer) and an output layer.

Input layer

We passed the following arguments to the keras.layers.InputLayer() function:.

  • input_shape: dimensionality of the input layer. In this case, the dimensions are the shape of each example (image) in X_train.

Hidden layers

  • Convolutional layers:

    As stated above, when constructing a convolutional hidden layer, we composed a 2D convolution, followed by a batch normalization, followed by an activation function.

    1. 2D convolution:
      • We used the Conv2D class to create each convolutional layer. For more information, consult the Keras online Conv2D class documentation. Conv2D() requires an argument specifying the number of convolutional filters in the layer. Layers closer to the input use fewer filters, while deeper layers use more filters to learn increasingly complex features.Therefore, we chose 16 filters for the first layer, 32 filters for the second layer, 64 filters for the third layer, and 128 filters for the fourth layer. Each layer uses a kernel size of 3, meaning each filter has a dimension of 3×3.

    1. Batch normalization:

      • Batch normalization is used to stabilize and speed up training by keeping the layer outputs on a consistent scale. For batch normalization, we used the BatchNormalization class. For more information, consult the Keras online BatchNormalization class documentation.

    2. Activation Function:

      • For the activation function, we used the ReLU class. For more information, consult the Keras online ReLU class documentation.
  • Pooling layer:
    • We used the GlobalAveragePooling2D class to create the pooling layer. For more information, consult the Keras online GlobalAveragePooling2D class documentation.

Output Layer:

The output layer is a single dense layer. We will have one node per class. We have ten classes (digits from 0-9). Therefore, when creating the output later, we specified 10 units.

Step 2. Define the Optimization Algorithm¶

As our optimizer we will use stochastic gradient descent (SGD) with a learning rate of 0.1.

In [13]:
sgd_optimizer = keras.optimizers.SGD(learning_rate=0.1)

Step 3. Define the Loss Function¶

Since we are working with a multi-class classification problem, we will use a sparse categorical cross entropy loss function using keras.losses.SparseCategoricalCrossentropy(). This is an extension of the categorical cross entropy loss function that is used when there are two or more label classes and the labels are integers. For more information, consult the online SparseCategoricalCrossentropy documentation.

In [14]:
loss_fn = keras.losses.SparseCategoricalCrossentropy(from_logits=True)

Step 4. Compile the Model¶

Now we will package the network architecture with the optimizer and the loss function using the cnn_model.compile() method. We will specify accuracyas the evaluation metric since we would like to obtain the accuracy score.

In [15]:
cnn_model.compile(optimizer=sgd_optimizer, loss=loss_fn, metrics=['accuracy'])

Part 5. Fit the Model to the Training Data¶

We can now fit the CNN model to the training data. Since there are 60,000 training examples and nearly 100,000 parameters to fit, this may take a while to run. Therefore, we will only choose one epoch.

Note: This may take a while to run.

In [16]:
num_epochs = 1 # Number of epochs

t0 = time.time() # start time

history = cnn_model.fit(X_train, y_train, epochs=num_epochs)

t1 = time.time() # stop time
